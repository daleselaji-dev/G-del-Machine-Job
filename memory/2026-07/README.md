# July 2026 Working Cache

Status: monthly working cache

## Purpose

This directory stores lightweight July 2026 working state for the daily GitHub-grounded job-search loop.

It is not a raw archive. It should preserve only information that helps the next daily run continue intelligently.

## Files

- `dashboard_state.md` — current dashboard state.
- `daily_deltas.md` — compressed changes from each daily run.
- `company_queue.md` — companies waiting for L0, L1 or L2 review.
- `evidence_gaps.md` — unresolved evidence gaps and how to resolve them.
- `workflow_improvements.md` — proposed improvements to the loop.

## Cache rules

Keep:

- current direction priorities;
- companies with active next actions;
- role leads with current evidence;
- unresolved validation questions;
- changes that alter future ranking or search direction;
- workflow issues that recur.

Do not keep:

- raw search dumps;
- every company reviewed;
- old anonymous reviews without decision value;
- unsupported speculation;
- temporary chat text.

## Review cadence

The cache should be cleaned weekly. Items should be promoted to durable memory only when they change future routing, ranking, application strategy or candidate evidence.
